# Reconnaissance de Locuteur en Langue Yemba (Cameroun)

Ce dépôt contient une implémentation en Python d’un système de reconnaissance de locuteur pour la langue Yemba du Cameroun. Le parcours suit les étapes suivantes:
1. Introduction
2. Projet
3. Données
4. Prétraitement des données
5. Modélisation et Interprétation
6. Déploiement
7. Conclusion

## Description
Ce projet vise à identifier le locuteur à partir d’enregistrements audio en Yemba. Il couvre:
- la collecte et l’annotation des données,
- le prétraitement audio ,
- l’extraction de caractéristiques ( MFCC...),
- la modélisation et l’évaluation ,
- un script de déploiement simple pour des prédictions en temps réel.

## NOTE: le jeu de données étant tres lourd, vous pouvez vous le rocurer a l'adresse:  "https://data.mendeley.com/datasets/74p9d5frg3/2"
